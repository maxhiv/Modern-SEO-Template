Prerequisites

Please follow below steps to install and setup all prerequisites:

  • PHP ~v8.0
  • Xampp (For Windows)

    Make sure to have latest version of xampp installed & running on your computer. If you already have Xampp installed on your computer, you can skip this step.

Installation

To setup, follow the below mentioned steps:

For Windows OS:

  • Unzip your project and place it inside the xampp/htdocs folder.
  • Start the Apache server from Xampp panel
  • Visit localhost/<project-name>/ after running Apache server locally

For Linux or macOS:

Command Description
php -S 127.0.0.1:8000 Starts the PHP server on 127.0.0.1:8000 address
